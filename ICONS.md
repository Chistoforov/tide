# Иконки для PWA

Иконки для PWA уже созданы и находятся в директории `/public/`:
- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей
- `icon.svg` - исходный SVG файл

## Регенерация иконок

Если нужно изменить дизайн иконки:

1. Отредактируйте файл `/public/icon.svg`
2. Запустите команду:
   ```bash
   npm run generate-icons
   ```

Это автоматически создаст PNG иконки нужных размеров из SVG.

## Альтернативные способы генерации

### Онлайн генераторы:
- https://realfavicongenerator.net/ - загрузите одну иконку, получите все размеры
- https://www.pwabuilder.com/imageGenerator - специализированный генератор PWA иконок
- https://favicon.io/ - простой генератор с эмодзи/текстом

### Рекомендации по дизайну:
- Используйте простой дизайн с волной или символом прилива
- Фон должен быть синим (#3b82f6) или прозрачным
- Иконки должны быть квадратными
- Для лучшей видимости используйте контрастные цвета
