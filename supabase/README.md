# Supabase Миграции

## Установка таблицы

Для создания таблицы `tide_data` в вашем проекте Supabase:

1. Откройте SQL Editor в панели управления Supabase
2. Скопируйте содержимое файла `migrations/001_create_tide_data_table.sql`
3. Выполните SQL запрос

Или используйте Supabase CLI:

```bash
supabase db push
```

## Структура таблицы

Таблица `tide_data` содержит:
- `id` - уникальный идентификатор записи
- `fetched_at` - время получения данных от API (используется для сортировки)
- `raw_data` - полные данные от Stormglass API в формате JSON
- `created_at` - время создания записи в базе данных

## Настройка RLS (Row Level Security)

Если вы используете RLS, убедитесь, что сервисный ключ имеет доступ к таблице. 
Для серверных операций через service role key RLS обычно не применяется.
